# QR Code Generator

A modern, feature-rich, and completely free QR Code Generator built with vanilla JavaScript and Tailwind CSS. Create professional, customizable QR codes for URLs, WiFi, contacts, payments, and more directly in your browser with no registration required.

## Live Demo

https://rakibalom.com/tools/qr-code-generator

## Features

### 12+ QR Code Types Supported

- URL - Website links for marketing campaigns and product pages
- Text - Plain text messages and quick notes
- WiFi - Network credentials for guest access in cafes and offices
- Email - Pre-filled emails for contact forms and support requests
- Phone - Direct dial numbers for customer service
- SMS - Text message templates for marketing opt-ins
- vCard - Digital business cards for networking
- Location - GPS coordinates for events and store locations
- Event - Calendar invitations for conferences and meetings
- Social - Social media profiles for follow buttons
- App Store - Mobile app links for downloads
- Payment - UPI, PayPal, Bitcoin, Ethereum for contactless payments

### Customization Options

- Custom foreground and background colors
- Adjustable dimensions from 150px to 350px
- Export as PNG or SVG formats
- High-resolution output suitable for print

### Privacy and Security

- Client-side generation, your data never leaves your browser
- No server storage, QR codes are generated locally
- No registration required

## Technologies Used

- HTML5 with semantic markup
- Tailwind CSS with custom glassmorphism design
- Vanilla JavaScript, no frameworks
- QRCode.js library for generation
- Schema.org structured data for SEO

## Installation

### Direct Download

Download the index.html file and open in any modern web browser.

### Clone Repository

git clone https://github.com/rakibalom/qr-code-generator.git
cd qr-code-generator

### Local Server

Using Python:
python -m http.server 8000

Using Node.js:
npx serve .

Then visit http://localhost:8000

## Usage Guide

1. Select Type - Choose from 12+ QR code types
2. Enter Details - Fill required information
3. Customize - Adjust colors and size
4. Generate - Click the Generate QR Code button
5. Download - Save as PNG or SVG or copy to clipboard
